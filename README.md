## 1. Opis projektu
Aplikacja klasy enterprise służąca do zarządzania ankietami akademickimi.

System umożliwia tworzenie oraz analizę ankiet dotyczących wykładowców.

Projekt został wykonany w ramach przedmiotu „Tworzenie aplikacji klasy enterprise”.

## 2. Funkcjonalności
- Zarządzanie ankietami
- Obsługa wykładowców
- Zarządzanie przedmiotami
- Tworzenie i edycja pytań

## 3. Technologie
- Java 21
- Spring Boot
- Spring Data JPA
- Spring Web
- H2 Database
- Gradle
